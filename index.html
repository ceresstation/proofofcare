Frequently asked questions

Q: Why should I use this site?
A: Modern computers are overpowered for most of the tasks that we perform on a regular basis (e.g. web browsing). As a result, there is a lot of unutilized processing power in most computers. By using Mine For Charity, you can donate those unutilized resources to mine cryptocurrency for charity.
Q: How does this work?
A: The Mine For Charity website includes a JavaScript miner for the Monero cryptocurrency. Simply by leaving this site open in your browser, you will be contributing to a mining network where earned cryptocurrency will be donated to charity.
Q: How much of the cryptocurrency mined gets donated?
A: As much as possible. We take a small portion for server fees and administrative costs. The rest gets donated to charity.
Q: Which cryptocurrency is being mined?
A: We mine Monero. The algorithm to mine Monero is compute heavy and designed to run well on CPUs. Unlike other cryptocurrencies like Bitcoin and Ethereum, which are 10,000x faster to mine using GPUs, Monero only has a 2x benefit when being mined on a GPU. This makes Monero a good choice for JavaScript based mining in the browser.
Q: Where does the cryptocurrency get donated?
A: We currently donate to The Water Project. As we grow, we will look into expanding to other charities and allowing users to specify a charity for their contributions.
Q: This site is making my computer run slow. What can I do?
A: You can lower the amount of processor being used for mining by adjusting the “Allowed CPU” slider. This setting changes the amount of processor time that miner uses. For example, if you set the Allowed CPU to 60%, the miner will use 60% of your CPU and leave 40% available for other tasks.
